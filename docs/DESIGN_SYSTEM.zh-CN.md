# AIOS Material Console 设计系统

AIOS Material Console 是 AIOS Control Center 的本地控制台风格。界面参考 Google Material 3、Google Workspace 与 Google Cloud Console 的产品感：浅色、克制、可扫描、面向操作，不做营销页和密集后台表格。

## 字体

- 全局使用系统无衬线字体：`ui-sans-serif`、Apple 系统字体、`Segoe UI`、`PingFang SC`、`Hiragino Sans GB`、`Microsoft YaHei`、本地可用的 `Noto Sans SC`。
- 不加载远程字体。
- 中文正文保持 14-16px，行高 1.45-1.65。
- 标题只用于层级和定位，不使用过大的营销式字号。
- 技术名、路径、命令和包名使用等宽字体，并允许自动换行。

## 色彩角色

- Primary：清晰的 Material 蓝，用于选中态、主操作、焦点和关键数字。
- Secondary：青绿，用于本地、只读和可用状态。
- Background：柔和中性浅灰蓝，降低长时间扫描疲劳。
- Surface：白色或近白色，用于导航、内容区、卡片和检查器。
- Text Primary：深灰黑，不使用纯黑压满页面。
- Text Secondary：中灰，用于说明、路径上下文和次级状态。
- Success / Warning / Error：分别表示低风险、中风险/警告、高风险。

## 表面与阴影

- 页面为 100dvh 控制台，不允许 body 或整页滚动。
- 左侧导航、中心工作区和右侧检查器是主要表面。
- 卡片使用软边界、轻阴影或低对比描边，避免硬线框堆叠。
- 常规容器圆角 20-28px；小控件圆角可更紧凑。
- 不使用随机渐变、玻璃拟态堆叠或强装饰背景。

## 间距

- 使用 8px 间距网格。
- 控制台外边距 12-16px。
- 面板内部间距 12-20px。
- 卡片内边距 14-18px。
- 信息组之间留出明显间隔，避免表格行密度。

## 卡片

- 卡片用于资源、指标、时间线和检查器块。
- 每张资源卡必须展示中文标题、原始技术名、资源类型、状态、风险、一行中文说明、路径/来源预览和紧凑操作。
- 卡片高度保持对齐，但说明文字最多两行。
- 不把卡片嵌套在另一张卡片中。
- 表格只能作为二级调试/详情区域使用，不能作为主要资源展示。

## 导航

- 使用 Material 导航 rail 或紧凑 Tabs。
- 模块包含：总览、技能、MCP、脚本、报告、项目包、策略、验证器、旧入口。
- 导航项必须显示中文模块名和数量。
- 选中态使用 primary 蓝背景与清晰对比。

## 搜索与命令栏

- 顶部产品头部包含搜索/命令栏。
- 占位文案使用中文。
- 搜索范围包括中文标题、原始技术名、路径、风险、状态和说明。
- 搜索是筛选，不执行命令。

## 风险徽章

- 低风险：绿色，表示只读元数据或低影响资源。
- 中风险：琥珀色，表示需要人工确认边界、凭据占位或执行方式。
- 高风险：红色，表示高影响执行、写入或敏感暴露风险。
- 同一行不堆叠过多彩色徽章；优先展示状态和风险两个主徽章。

## MCP 卡片

- MCP 资源按本地、远程、npx 拉取、需要凭据分组。
- 只展示命令名、传输、来源路径和环境变量名数量，不展示变量值。
- `@latest` 和 npx 拉取必须标记为“可能拉取”或“包含 @latest”。
- 页面不得启动或连接 MCP 服务。

## 技能卡片

- 技能卡片必须显示中文能力标题和原始技能名。
- Codex、Agents、Claude、运行时视图和注册表应分组展示。
- 复制提示词按钮保留英文提示词正文，但按钮和解释文案使用中文。
- 不把项目本地技能复制到全局入口。

## 报告与时间线

- 报告使用时间线卡片展示，不使用原始文件列表。
- 每张报告卡展示中文标题、原始文件名、状态、风险、摘要和路径。
- 时间线只读，不写入报告目录。

## 动效

- 使用轻量 Material motion：模块卡片进入、模块切换、检查器切换、复制反馈。
- 动效只使用透明度与 transform。
- 必须尊重 `prefers-reduced-motion`。
- 禁止滚动动画与 ScrollTrigger。

## 禁止项

- 密集后台表格作为主界面。
- 随机渐变和装饰性背景。
- 不一致卡片尺寸。
- 面向用户的英文 UI 标签。
- 整页滚动。
- 全局写入。
- 恢复或重建 Codex 自动化。
- 修改全局技能入口、MCP、认证、供应商、模型、launchd、浏览器扩展、凭据或环境配置。
