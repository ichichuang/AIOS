# 产品验收标准

## 文档验收

- 产品文档位于 `docs/product/`。
- **必须包含并优先引用 `docs/product/00-aios-desktop-product-definition.zh-CN.md` 作为最高权威产品定义。**
- 至少包含产品定位、用户需求、首次使用流程、信息架构、术语指南、技能库需求、MCP 工具库需求、隐私与安全需求、产品验收标准。
- 文档使用简体中文。
- 文档默认面向普通用户。
- 文档明确说明本阶段不做实现，不改代码。
- 文档明确说明 Scripts、Reports、Policies、Validators、Legacy、原始查找诊断和内部调试概念不是一级产品模块。

## 界面简单性验收

- 主导航只有：首页、技能、MCP、高级。
- 首页必须 visibly 分离以下模块：Hero、本地整理内容概览、项目能力状态、技能入口、MCP 入口、本地隐私说明。P6C 阶段首页不显示“需要处理”模块。
- 项目能力概览必须诚实说明项目级 AI 能力组织在 P6C 阶段暂不可用；不得使用 `resourceCorpus.projectMap`、路径或推断名称渲染项目。
- 首屏能看到已整理的本地技能数量、已记录的 MCP 服务数量和工具名称线索数量；P6C 阶段不显示需要处理的数量。
- 普通用户不需要理解内部工程词汇也能完成首次查找。
- 高级内容不会出现在首页默认区域。
- 任意页面都不能把脚本、报告、旧入口、检查器或内部诊断作为一级模块展示。

## 技能数量准确性验收

- 技能默认浏览按任务和功能分类。
- 技能来源只作为筛选、详情来源说明和高级来源视图。
- 首页技能总数必须等于技能库当前可见来源的去重结果。
- 重复技能不能重复计入“可用技能”。
- 来源不明项目不能计入“可用技能”，除非已经确认是完整技能。
- 已损坏技能可以计入“需要处理”，但不能计入“可用技能”。
- 技能详情必须能说明该技能来自哪里。
- 技能详情必须能说明哪个 AI 工具可以使用它。
- 技能详情必须提供使用方式或说明为什么暂时无法判断。

## MCP 可见性验收

- MCP 默认浏览按服务展示。
- MCP 状态和来源只作为筛选、详情状态和高级来源视图。
- 首页显示 MCP 服务数和 MCP 工具数。
- MCP 页面显示服务列表和每个服务的工具列表。
- MCP 页面显示配置来源、用途、问题和人工处理建议。
- 无法读取工具列表时，页面必须说明不会启动服务来获取更多内容。
- 页面不能显示密钥值、令牌值或环境变量值。
- 页面不能提供启动服务或调用工具的操作。

## 普通用户一键流程验收

- 首次启动不自动查找。
- 首页有明确的“开始查找”按钮。
- 点击开始前说明会查找什么、不会做什么。
- 查找过程中显示简单进度。
- 查找结束后显示技能数量、MCP 数量和需要处理的数量。
- 没有结果时，引导用户手动选择文件夹。

## 高级开发者来源分组验收

- 高级页可以按来源查看技能和 MCP。
- 来源至少支持全局来源、项目来源、Codex、Claude、手动选择的文件夹、来源不明。
- 高级页可以显示路径提示，但必须隐藏敏感部分。
- 高级页可以显示跳过项和问题摘要，但不能显示原始日志或密钥。
- 高级页不能提供自动修改配置、安装依赖、执行脚本或启动 MCP 的操作。

## 术语合规验收

- 普通用户文案使用“本机”“已整理”“已记录”“项目”“技能”“MCP 服务”“能做什么”“适合什么时候使用”“可用于”“来源”“来自本地配置记录”“未进行实时连接验证”“工具名称线索”“暂未整理项目级 AI 能力”“暂无可整理的内容”等简单词。
- 禁用“资源语料”“corpus”“scanner”“资源检查器”“Token 压力”“元数据质量”“数据库记录”“原始资源”“runtime”“schema”“command contract”“raw ID”“absolute path”“internal status”等工程词汇出现在普通用户主界面。
- 不得把本地聚合数据描述为全电脑或全局范围。
- 不得把 MCP `sourceLabel` 描述为应用兼容性。
- 不得把 MCP 工具名称线索描述为已验证工具。
- MCP 第一次出现时有中文解释。
- 技能第一次出现时有中文解释。
- 禁用术语只允许出现在术语指南的禁用表、信息架构的移除说明或开发者补充中。
- 任何面向普通用户的标题、按钮、空状态、错误文案，都不能使用难懂工程词汇。

## 隐私与安全验收

- 文档明确说明结果只保存在本机。
- 文档明确说明不上传查找结果。
- 文档明确说明不读取密钥、令牌、密码、Cookie、登录会话或环境变量值。
- 文档明确说明不执行脚本、不启动 MCP 服务、不调用 MCP 工具。
- 文档明确说明清空本地记录不会删除用户文件。

## 本阶段停止线

完成这些文档后停止。不得开始 UI、扫描、存储、构建、工作流或发行实现。
