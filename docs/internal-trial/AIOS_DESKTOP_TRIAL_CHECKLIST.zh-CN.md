# AIOS Desktop 可信内部试用检查清单

基线 commit：`23e71f226cdd99ffc1973954f9d70ca0058c0474`

试用结论只适用于可信内部 Mac packaged trial，不适合公开发布。

## 环境

- [ ] 使用 Mac 本地测试。
- [ ] 使用匹配芯片架构的内部 unsigned 构建。
- [ ] 测试文件夹是合成 fixture 或小型安全项目。
- [ ] 未扫描 home root、`/Users`、`/Volumes`、`/System`、`/Library`、整块磁盘、生产仓库、凭据目录或浏览器 profile。

## 启动与导航

- [ ] App 能启动。
- [ ] 主导航显示：首页 / 技能 / MCP / 高级。
- [ ] 首页引导能打开。
- [ ] 点击“开始查找”后能看到安全边界说明。

## 查找流程

- [ ] 选择文件夹不会自动扫描。
- [ ] 需要手动点击 start scan / 开始查找才会运行。
- [ ] 扫描开始时有清楚反馈。
- [ ] 运行中状态可见。
- [ ] 完成状态可见。
- [ ] 失败状态可见，且失败说明可理解。
- [ ] 完成摘要出现。

## 技能结果

- [ ] 技能行能渲染。
- [ ] 技能搜索或筛选可用。
- [ ] 技能详情能打开。
- [ ] 技能详情说明用途、来源、可用工具、使用方式和状态。
- [ ] 技能详情不执行技能、不修改文件。

## MCP 结果

- [ ] MCP 服务行能渲染。
- [ ] MCP 工具行能渲染。
- [ ] MCP 详情能打开。
- [ ] MCP host 只显示 hostname-only。
- [ ] 环境变量输出只显示名称，不显示值。
- [ ] 没有出现 fake secret-like strings。
- [ ] 没有预期或要求 MCP 进程启动。
- [ ] 没有预期或要求 endpoint connection。
- [ ] 没有预期或要求 MCP tool call。
- [ ] MCP 信息被理解为静态/未验证元数据；不承诺 MCP 实时运行状态。

## 隐私反馈检查

- [ ] 反馈中不含密钥、token、密码、Cookie、登录会话、环境变量值或客户数据。
- [ ] 截图已去除敏感内容。
- [ ] 日志如需提供，已人工审阅并脱敏。
- [ ] 未提交原始私有路径；必要路径已截断或泛化。
